What is this site for?

Koe started as a simple text-to-speech tool for Japanese and Turkish, built on Microsoft's neural voices so you can hear text read aloud and download the audio, no account required.

It has since grown into a small language-learning corner too — the Turkish Lessons section pairs narrated stories with the text on the page, so you can read along while you listen and pick up new words in context.

Runs on Microsoft Edge neural voices · no account needed.
